
#### 1.3.3 页面语言
```html

```
#### 1.3.4 添加样式表
```html

```
#### 1.3.5 添加JavaScript
```html

```
#### 1.3.6 最终结果
```html
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="utf-8">
<title>A Tiny HTML Document</title>
<link rel="stylesheet" href="styles.css">
<script src="scripts.js"></script>
</head>
<body>
<p>Let's rock the browser,HTML5 style.</p>
</body>
</html>
```

- 用于标识文本的语义元素 mark、time、wbr

- Canvas
